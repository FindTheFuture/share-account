# 修改账单列表页面功能

## 1. 显示时间范围在时间标签上

* 在 `data` 中添加 `dateRangeText` 变量用于存储显示的时间范围文本

* 修改 `setDefaultDateRange` 方法，计算并设置 `dateRangeText`

* 修改时间标签的文本绑定，显示 `dateRangeText` 而不是固定的 "时间"

* 在 `onDatePickerChange` 和 `confirmDateRangeSelection` 方法中更新 `dateRangeText`

## 2. 调整重置和导出标签到下一行

* 修改 `top-toolbar` 的 CSS 布局，将其改为 `flex-wrap: wrap`

* 为重置和导出标签添加特殊的 CSS 类，使其占据整行或换行显示

* 调整其他标签的宽度和间距，确保布局美观

## 3. 增加排序功能

* 在 `data` 中添加排序相关变量：`selectedSortOption`、`sortPopupVisible` 等

* 在工具栏中添加排序标签

* 创建排序弹窗组件，包含4个排序选项：1、时间由近到远，2、时间由远到近，3、金额由大到小，4、金额由小到大

* 实现排序标签点击事件，显示弹窗

* 实现排序选项点击事件，更新排序参数并重新请求数据

* 修改 `loadBillList` 方法，将排序参数传递给后端，默认传递1

## 4. 支持后端排序参数

* 在 `billListParams` 中添加 `sortBy`字段，并传递4个排序选项的id值，默认传递1

* 当用户选择排序选项时，更新这些字段

* 在向后端发送请求时，包含排序参数

* 后端根据这些参数调整 SQL 查询的 ORDER BY 子句
